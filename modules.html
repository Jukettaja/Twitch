Twitch API

The basis of the entire project. The Twitch API is a fluent api, is based on an internal builder, so you don't have to worry about the implementation.

Furthermore, the entire oauth 2 mechanism (the way you authenticate and authorize users) is implemented and ready for you to use.

Twitch Boot Starter

Want your project to easily integrate with Spring Boot? We provide you with a module you just have to include using your build manager and voila, you now have a Spring Boot application which includes our Twitch API. No extra setup required!
